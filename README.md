# nextcrawler
Next Crawler 是使用Playwright + Next.js + Prisma等主流技术搭建网页数据采集器，通过可视化的UI进行配置，即可周期性的通过Playwright驱动浏览器爬取网页数据。内置支持 使用mozilla/readability智能识别网页正文； 可配置的正文图片自动下载功能； 可以选择的文件下载功能，支持PDF、MP3、MP4、等多种格式； 可以配置的多字段解析； 可以配置的Playwright基本动作； 支持定时任务，并且可以配置周期采集数量； 支持采集模板，并且支持导入导出； 支持代理； 支持错误日志； 可选的基本用户认证管理。
